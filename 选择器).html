7. 选择器速查表

选择器 名称 示例 描述
* 通用选择器 * { margin: 0; } 选择所有元素
E 元素选择器 p { color: red; } 选择所有p元素
.class 类选择器 .highlight { background: yellow; } 选择class="highlight"的所有元素
#id ID选择器 #nav { width: 100%; } 选择id="nav"的元素
E F 后代选择器 div p { margin: 5px; } 选择div内所有的p元素
E > F 子选择器 ul > li { list-style: none; } 选择E的直接子元素F
E + F 相邻兄弟选择器 h1 + p { margin-top: 0; } 选择紧接在E后的F元素
E ~ F 一般兄弟选择器 h1 ~ p { color: gray; } 选择E后面的所有F兄弟元素
[attr] 属性存在选择器 [title] { border: 1px solid; } 选择具有attr属性的元素
[attr=value] 属性值选择器 [type=text] { width: 100%; } 选择attr属性值等于value的元素
:hover 伪类选择器 a:hover { color: red; } 鼠标悬停时的状态
::before 伪元素选择器 p::before { content: "> "; } 在元素内容前插入内容

8. 优先级权重

CSS选择器的优先级（从高到低）：

  1. !important - 最高优先级（不推荐过度使用）
  2. 内联样式 - style="" （权值: 1,0,0,0）
  3. ID选择器 （权值: 0,1,0,0）
  4. 类选择器/属性选择器/伪类 （权值: 0,0,1,0）
  5. 元素选择器/伪元素 （权值: 0,0,0,1）
  6. 通用选择器 （权值: 0,0,0,0）

例如：#nav .menu ul li.active 的权值是 0,1,1,3
